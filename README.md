# 🏰 Projet Mini-RPG - Le Donjon de Naheulbeuk

Bienvenue dans le dépôt de mon tout premier projet de jeu vidéo ! C'est un mini-RPG textuel développé en Python, inspiré de l'univers déjanté et humoristique du **Donjon de Naheulbeuk**.

Ce projet s'inscrit dans le cadre de mon apprentissage autodidacte de la programmation et de Python.

## 🚀 Fonctionnalités actuelles
- **Création de personnage :** Choix des statistiques et personnalisation.
- **Mécaniques de jeu :** Gestion des mécaniques de base et du gameplay (en cours de développement).

## 🛠️ Technologies utilisées
- **Langage :** Python 3
- **Outils :** Git & GitHub pour le versioning.
